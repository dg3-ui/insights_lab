# Render Examples — Read the Date Before Imitating

> **Status**: note added 2026-06-12 after the California-build feedback intake (`docs/learning/logs/2026-06-12_california_build_feedback_intake.md`).

The two `*_001.html` renders are the **Phase-2 acceptance records** (2026-06-12, pre-feedback). They proved the brand pipeline (fonts · anatomy · grounding) and stay as records — but **three conventions changed after them**, so do not imitate these parts:

| In the 001 renders | Current rule (post-intake) |
|---|---|
| `CONFIDENCE: LOW` badge + footer label on the **public** blog | public renders carry calibration as *posture* ("a forward, directional read"); the explicit label is **internal-only** (`output_contracts.md` §3, `brand.md` §4) |
| "What we are not claiming" negation box on the public blog | public caveat box is the *honest caveat* — what's still resolving, what would change the read (negation list stays internal) |
| single-plant CF chart leading a **top-down** piece | top-down figures lead at fleet/region scope; a single-plant series appears only as a **labeled exemplar inset** until R2/R8 ship (`_craft/plot_generation.md` §4) |

The next composed test run (plan Phase 6b) produces the current-contract exemplar (`applied_insight_002` + its renders); these files then stand as the before/after pair.
