# docs/discussion — Open Discussions (upstream of plans)

> **Status**: opened 2026-06-13. The home for **exploratory discussions** — the thinking we do *before* a plan is warranted.

## What a discussion is (and how it differs from a plan)

```text
DISCUSSION (here)                          PLAN (docs/plans/)
open-ended · no phase ladder · iterated    committed · phased · executed one phase at a time
"what should we do, and how should we      "here is the decided work, in order"
 categorize / sequence / frame it?"
a draft to argue with                      a contract to execute
        │  resolves into decisions
        └──────────────► spawns a plan in docs/plans/ (or amends canon)
```

A discussion is a **draft to iterate on**, not a decision and not canon. It gathers options, grounds them, and surfaces the real open questions. When a discussion resolves, its decisions either become a plan (`docs/plans/`) or land as canon amendments — and the discussion stays as the record of *why*.

House doc style applies (`Status` blockquote · ASCII + tight tables · `See also` footer). Nothing in a discussion binds until it's promoted to a plan or canon.
