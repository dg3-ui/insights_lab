# docs/plans/done — Completed & Superseded Plans (the archive)

> **Status**: opened 2026-06-13. The archive for plans that have **closed** — every phase done, or the plan superseded by a newer one.

## What Lands Here

```text
a plan is DONE / SUPERSEDED
        │  mark its final status line (DONE <date> · or SUPERSEDED by <plan>)
        │  git mv the file here verbatim — a relocation, NOT a rewrite (history preserved)
        ▼
docs/plans/done/<original-filename>.md      + the plans/README index keeps a pointer row
```

This keeps the top-level `plans/` showing only **live** work (ACTIVE · QUEUED · PAUSED). A closed plan stays the record of *what was decided and why* — it is never deleted or edited after closure, only moved.
